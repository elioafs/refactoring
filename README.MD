# Refactoring

Refactoring it's a project for training refactoring skills

## Installation

Use the package manager [mvnw] on project.

```bash
./mvnw clean package
```

```cmd
mvnw.cmd clean package
```

## Contributing
Each one create a branch and make your refactor.

You can make tests as appropriate.
